## 🎮 使用提示

1. **精灵表(Sprite Sheet)**: 下载的资源通常是精灵表，需要在游戏引擎中分割成单个帧
2. **支持的引擎**: Unity、Godot、GameMaker、Construct、RPG Maker等
3. **帧大小**: 查看每个资源的说明，了解单个帧的尺寸
4. **授权检查**: 使用前确认每个资源的授权协议
