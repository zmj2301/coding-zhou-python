## 🎮 使用方法

1. 精灵表(Sprite Sheet)需要在游戏引擎中分割成单个帧
2. 支持的引擎: Unity、Godot、GameMaker、Construct、RPG Maker
3. 冰火主题: 可以给角色重新着色（蓝色=冰系，红色=火系）
