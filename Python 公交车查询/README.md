
# 湘潭公交查询系统

一个简单易用的湘潭市公交车查询工具，支持站点搜索、线路查询和实时到站信息显示。

## 功能特点

- 🔍 站点搜索：支持关键词搜索湘潭公交站点
- 🚌 线路查询：查看所有公交线路及详细信息
- ⏱️ 实时到站：模拟显示公交车预计到达时间
- 💻 命令行界面：简洁友好的交互界面

## 项目结构

```
湘潭公交查询系统/
├── bus_query.py      # 核心查询模块
├── main.py           # 主程序入口
├── test.py           # 测试脚本
├── requirements.txt  # 依赖文件
├── data/
│   └── bus_data.json # 公交数据
└── README.md         # 说明文档
```

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行程序

```bash
python main.py
```

### 3. 运行测试

```bash
python test.py
```

## 使用说明

程序启动后，会显示操作菜单：

```
==================================================
           湘潭公交查询系统
==================================================
1. 搜索站点
2. 查看所有线路
3. 查询线路详情
4. 查询站点实时到站
5. 退出
==================================================
```

选择对应的功能编号即可使用。

## 数据说明

当前系统包含湘潭市 5 条主要公交线路的模拟数据：
- 1路：火车站 → 半边街
- 2路：公交西站 → 鹤岭镇
- 3路：新城社区 → 涟水桥
- 5路：新城社区 → 砂子岭
- 6路：观湘门 → 湘潭大学

## 注意事项

⚠️ 当前使用的是模拟数据，实时到站信息为随机生成，仅供演示使用。
若需要接入真实数据，需要对接湘潭公交官方 API。

## 扩展开发

如需接入真实公交数据，可修改 `bus_query.py` 中的数据获取逻辑，对接官方 API。
