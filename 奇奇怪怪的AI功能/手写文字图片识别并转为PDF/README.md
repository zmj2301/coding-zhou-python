# 像素风机甲大战 (Pixel Mech Battle)

## 游戏简介

一款复古像素风格的双人本地对战游戏。两位玩家各自控制一台机甲，在废弃的工业战场上进行激烈对战。

## 游戏特色

- **复古像素风格**: 采用 8-bit 像素艺术风格，每个角色由 4x4 像素块组成
- **双人对战**: 支持两位玩家同时操作各自机甲进行对战
- **战斗系统**: 包含攻击、防御、技能特效等完整的战斗机制
- **粒子特效**: 攻击火花、防御护盾等视觉反馈
- **流畅动画**: 60 FPS 游戏循环，支持角色移动动画

## 角色设定

### 🔴 红色机甲 (Red Fury)
- 特点：重型机甲，高攻击力
- 生命值：100 HP
- 攻击力：15 点/次
- 移动速度：较慢

### 🔵 蓝色机甲 (Blue Storm)
- 特点：敏捷机甲，高机动性
- 生命值：80 HP
- 攻击力：10 点/次
- 移动速度：较快

## 操作说明

### 玩家1 (红色机甲)
- **W/A/S/D**: 上下左右移动
- **F**: 普通攻击
- **G**: 防御姿态

### 玩家2 (蓝色机甲)
- **↑/←/↓/→**: 上下左右移动
- **J**: 普通攻击
- **K**: 防御姿态

### 游戏控制
- **空格键**: 开始游戏
- **R**: 重新开始

## 游戏规则

1. 两个玩家控制各自的机甲
2. 通过移动接近对手并进行攻击
3. 防御可以减少 50% 受到的伤害
4. 防御时无法移动和攻击
5. 当任一机甲生命值归零时，游戏结束
6. 获胜者显示胜利画面

## 技术实现

### 核心技术
- HTML5 Canvas: 游戏渲染
- Vanilla JavaScript: 游戏逻辑
- CSS3: 界面样式
- Google Fonts: 像素风格字体

### 架构模块
- `InputManager`: 键盘输入管理
- `Mecha`: 机甲角色类
- `Particle`: 粒子特效系统
- `Game`: 游戏主循环和状态管理

## 运行方法

1. 直接在浏览器中打开 `index.html` 文件
2. 推荐使用 Chrome 或 Firefox 浏览器
3. 确保启用 JavaScript
