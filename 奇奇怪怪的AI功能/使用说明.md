# 手写文字OCR识别与PDF转换工具 - 使用说明

## 📦 快速开始

### 方法一：使用批处理脚本（推荐）

1. **双击运行 `一键安装.bat`** - 自动安装所有依赖库
2. **双击运行 `启动OCR工具.bat`** - 启动图形界面工具

### 方法二：使用命令行

```bash
# 安装依赖
D:\python\python.exe -m pip install Pillow reportlab paddlepaddle paddleocr

# 启动工具
D:\python\python.exe OCR转换工具.py
```

## 🎯 功能特点

- ✅ **OCR文字识别** - 使用PaddleOCR识别手写和印刷文字
- ✅ **PDF生成** - 将识别结果生成可搜索的PDF文档
- ✅ **图形界面** - 简单易用的GUI操作
- ✅ **批量处理** - 支持批量处理文件夹中的图片
- ✅ **格式保留** - 支持保留原始图片或仅保留文字

## 📋 支持的图片格式

- JPG / JPEG
- PNG
- BMP
- TIFF / TIF

## 🚀 使用步骤

### 1. 安装依赖

双击运行 `一键安装.bat`，等待安装完成。

依赖包括：
- Pillow - 图片处理
- ReportLab - PDF生成
- PaddlePaddle - 深度学习框架
- PaddleOCR - OCR识别引擎

### 2. 启动工具

双击运行 `启动OCR工具.bat`

### 3. 选择目录

- **输入目录**：选择包含图片的文件夹（如：`E:\桌面\P19-29`）
- **输出目录**：选择保存PDF的位置（默认在输入目录的output子目录）

### 4. 设置选项

- **启用OCR文字识别** - 勾选此项进行文字识别
- **在PDF中包含原始图片** - 勾选此项在PDF中同时保留原始图片

### 5. 开始转换

点击 **「🚀 开始转换」** 按钮，等待处理完成。

## ⚙️ 常见问题

### Q: 安装速度慢怎么办？
A: 脚本已使用清华大学镜像源，速度应该很快。如遇问题可手动安装。

### Q: 第一次运行很慢？
A: 第一次运行OCR时，PaddleOCR会自动下载模型文件（约100MB），请耐心等待。

### Q: 识别准确率不高？
A: 
- 确保图片清晰度足够
- 文字与背景对比度要高
- 尽量保持图片正向（不要旋转）

### Q: 可以只提取文字不保留图片吗？
A: 可以！取消勾选「在PDF中包含原始图片」即可。

## 📞 技术支持

如遇问题，请检查：
1. Python路径是否正确（D:\python\python.exe）
2. 是否已安装所有依赖库
3. 输入目录是否包含支持的图片文件
