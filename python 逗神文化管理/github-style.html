逗神归来 - 视频学习平台

一个设计精美的本地视频播放网站，采用极简设计风格，支持本地视频加载、分类筛选、搜索等功能。

功能特点

  • 极简设计 - 简洁优雅的用户界面
  • 文件夹扫描 - 自动扫描并分类本地视频文件
  • 搜索与筛选 - 按分类和关键词搜索视频
  • 响应式布局 - 支持桌面和移动设备
  • 视频播放 - 内置视频播放器
  • 暗色模式 - 自动适配系统主题

快速开始

  1. 直接在浏览器中打开 index.html
  2. 点击"选择文件夹"按钮，选择包含视频的文件夹
  3. 点击视频卡片即可播放

文件结构

逗神文化管理/
├── index.html          # 原应用主页面
├── index-app.html      # 简化版应用
├── github-style.html   # GitHub风格项目展示页
├── server.py           # Python后端服务器
├── README.md           # 说明文档

技术栈

  • HTML5 + CSS3 + JavaScript
  • Font Awesome 图标库
  • Google Fonts (Inter)
  • Python HTTP Server (可选)
