## 🚀 快速开始

### 方法一：直接打开（演示模式）
1. 双击 `index.html` 文件
2. 在浏览器中查看演示数据
3. 点击"选择文件夹"按钮可以上传本地视频

### 方法二：Python服务器（推荐，完整功能）

#### 启动方式：
1. **Windows一键启动**：双击 `启动服务器.bat`
2. **命令行启动**：
   ```bash
   cd "e:\coding-zhou\Python\python 逗神文化管理"
   python server.py
   ```

#### 访问网站：
- 打开浏览器，访问：`http://localhost:8000`
- 服务器会自动扫描 `e:\豆神归来文件资料` 文件夹

---

## 📁 文件夹配置

如果你的视频文件夹不在 `e:\豆神归来文件资料`，可以修改 `server.py` 中的路径：

```python
VIDEO_ROOT = Path(r"你的视频文件夹路径")
```

## 🛠️ 常见问题

### Q: 服务器启动失败怎么办？
A: 确保已安装Python 3.x，并检查视频文件夹路径是否正确。

### Q: 视频无法播放？
A: 请确保视频格式是浏览器支持的（.mp4、.webm等）。

### Q: 如何添加新视频？
A: 将新视频放入 `e:\豆神归来文件资料` 文件夹，刷新网页即可。

### Q: 可以修改网站样式吗？
A: 可以！直接编辑 `index.html` 中的 CSS 部分即可。
