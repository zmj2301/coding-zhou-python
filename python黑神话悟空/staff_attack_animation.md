# 黑神话悟空 - 劈棍/戳棍/立棍动画系统

## 动画配置说明

### 1. 蓄力动画（劈棍蓄力）
- **总帧数**: 6帧
- **帧间隔**: 12ms
- **动画时长**: 6 × 12 = 72ms
- **动画效果**: 力量逐渐积累，棍子举得越来越高
- **帧序列**: 劈棍蓄力1 → 劈棍蓄力2 → ... → 劈棍蓄力6

### 2. 攻击动画（劈棍攻击）
- **总帧数**: 3帧
- **帧间隔**: 8ms
- **动画时长**: 3 × 8 = 24ms
- **动画效果**: 爆发力十足的下劈，有冲击感
- **帧序列**: 劈棍攻击1 → 劈棍攻击2 → 劈棍攻击3

### 3. 蓄力动画（戳棍蓄力）
- **总帧数**: 3帧
- **帧间隔**: 12ms
- **动画时长**: 3 × 12 = 36ms
- **动画效果**: 力量逐渐积累，棍子前伸蓄力
- **帧序列**: 戳棍蓄力1 → 戳棍蓄力2 → 戳棍蓄力3（在2-3帧循环）

### 4. 攻击动画（戳棍攻击）
- **总帧数**: 7帧
- **帧间隔**: 8ms
- **动画时长**: 7 × 8 = 56ms
- **动画效果**: 快速前刺攻击，连续突进
- **帧序列**: 戳棍攻击1 → 戳棍攻击2 → ... → 戳棍攻击7

### 5. 蓄力动画（立棍蓄力）
- **总帧数**: 8帧
- **帧间隔**: 12ms
- **动画时长**: 8 × 12 = 96ms
- **动画效果**: 力量逐渐积累，棍子竖起蓄力
- **帧序列**: 立棍蓄力1 → 立棍蓄力2 → ... → 立棍蓄力8（在4-8帧循环）

### 6. 攻击动画（立棍攻击）
- **总帧数**: 5帧
- **帧间隔**: 8ms
- **动画时长**: 5 × 8 = 40ms
- **动画效果**: 从立姿向下猛劈，威力巨大
- **帧序列**: 立棍攻击1 → 立棍攻击2 → ... → 立棍攻击5

### 7. 操作方式

#### 劈棍攻击
- **按键**: 按住 **L键** 蓄力，松开释放攻击
- **流程**:
  1. 待机状态下按L → 开始蓄力
  2. 快速播放1-2帧，然后在3-6帧循环
  3. 按住L保持蓄力状态（3-6帧循环）
  4. 松开L → 播放3帧攻击动画
  5. 自动回到待机状态

#### 戳棍攻击
- **按键**: 按住 **I键** 蓄力，松开释放攻击
- **流程**:
  1. 待机状态下按I → 开始蓄力
  2. 快速播放1帧，然后在2-3帧循环
  3. 按住I保持蓄力状态（2-3帧循环）
  4. 松开I → 播放7帧攻击动画
  5. 自动回到待机状态

#### 立棍攻击
- **按键**: 按住 **O键** 蓄力，松开释放攻击
- **流程**:
  1. 待机状态下按O → 开始蓄力
  2. 快速播放1-3帧，然后在4-8帧循环
  3. 按住O保持蓄力状态（4-8帧循环）
  4. 松开O → 播放5帧攻击动画
  5. 自动回到待机状态

## 代码配置参数

### Wukong 类配置
```python
# 劈棍动画帧数
self.staff_attack_charging_frames = 6   # 蓄力帧数
self.staff_attack_strike_frames = 3    # 攻击帧数

# 戳棍动画帧数
self.poke_attack_charging_frames = 3   # 蓄力帧数
self.poke_attack_strike_frames = 7    # 攻击帧数

# 立棍动画帧数
self.stand_attack_charging_frames = 8  # 蓄力帧数
self.stand_attack_strike_frames = 5   # 攻击帧数

# 帧间隔（越大越慢）
self.standby_interval_dictionary = {
    '劈棍蓄力': 12,   # 蓄力帧间隔
    '劈棍攻击': 8,    # 攻击帧间隔
    '戳棍蓄力': 12,   # 蓄力帧间隔
    '戳棍攻击': 8,    # 攻击帧间隔
    '立棍蓄力': 12,   # 蓄力帧间隔
    '立棍攻击': 8     # 攻击帧间隔
}

# 劈棍状态控制
self.is_charging = False     # 是否在蓄力中
self.charging_held = False   # 是否按住蓄力键

# 戳棍状态控制
self.poke_is_charging = False  # 是否在蓄力中
self.poke_charging_held = False # 是否按住蓄力键

# 立棍状态控制
self.stand_is_charging = False  # 是否在蓄力中
self.stand_charging_held = False # 是否按住蓄力键

self.can_change_state = True  # 是否可以切换状态

# 位置微调
self.staff_attack_offset_x = 0  # 劈棍水平偏移
self.staff_attack_offset_y = 0  # 劈棍垂直偏移
self.poke_attack_offset_x = 0   # 戳棍水平偏移
self.poke_attack_offset_y = 0   # 戳棍垂直偏移
self.stand_attack_offset_x = 0  # 立棍水平偏移
self.stand_attack_offset_y = 0  # 立棍垂直偏移
```

## 状态机流转

### 劈棍攻击
```
待机 → 劈棍蓄力（按住L）→ 劈棍攻击（松开L）→ 待机
              ↓
         停在第3-6帧循环
```

### 戳棍攻击
```
待机 → 戳棍蓄力（按住I）→ 戳棍攻击（松开I）→ 待机
              ↓
         停在第2-3帧循环
```

### 立棍攻击
```
待机 → 立棍蓄力（按住O）→ 立棍攻击（松开O）→ 待机
              ↓
         停在第4-8帧循环
```

## 其他按键
- **WASD**: 移动
- **K键**: 轻攻击
- **J键**: 棍花
- **L键**: 劈棍蓄力/攻击
- **I键**: 戳棍蓄力/攻击
- **O键**: 立棍蓄力/攻击

## 调整建议
1. 调整动画速度: 修改 `standby_interval_dictionary` 中的值
2. 调整位置对齐: 修改 `staff_attack_offset_x/y` 或 `poke_attack_offset_x/y` 或 `stand_attack_offset_x/y`
3. 调整按键: 修改Game类中的按键码
