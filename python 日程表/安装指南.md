# 环境安装指南

## 依赖包安装

### 方法 1：使用 pip 安装
```bash
# 安装所有依赖
pip install -r requirements.txt

# 或者逐个安装
pip install PySide6==6.6.0
pip install pandas==2.2.0
pip install openpyxl==3.1.2
pip install zai==0.1.0
```

### 方法 2：使用虚拟环境（推荐）
```bash
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Windows
env\Scripts\activate
# Linux/Mac
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt
```

## 注意事项

1. **PySide6 安装**：如果安装失败，可能需要：
   - 更新 pip: `pip install --upgrade pip`
   - 安装 Visual C++ 运行时（Windows）

2. **pandas 安装**：pandas 会自动安装其依赖（如 numpy）

3. **Excel 文件**：确保 `456.xlsx` 文件与应用程序在同一目录

4. **图标文件**：确保 `combined.ico` 文件与应用程序在同一目录

## 运行程序
```bash
# 直接运行
python show_yourwindows.py

# 或使用打包后的 exe 文件
# 从 dist 目录运行 show_yourwindows.exe
```

## 故障排除

### 常见错误及解决方案

1. **ImportError: No module named 'PySide6'**
   - 解决方案：`pip install PySide6`

2. **ImportError: No module named 'pandas'**
   - 解决方案：`pip install pandas openpyxl`

3. **FileNotFoundError: [Errno 2] No such file or directory: '456.xlsx'**
   - 解决方案：确保 Excel 文件在正确的位置

4. **TclError: bitmap "combined.ico" not defined**
   - 解决方案：确保 icon 文件在正确的位置

5. **AttributeError: 'MessageFactory' object has no attribute 'GetPrototype'**
   - 解决方案：这是 TensorFlow/TensorBoard 兼容性问题，不影响本程序运行

6. **ModuleNotFoundError: No module named 'pkg_resources'**
   - 解决方案：`pip install setuptools==75.8.0`

## 系统要求

- Python 3.8 或更高版本
- Windows 7/10/11
- 足够的内存（推荐 4GB+）
- 足够的磁盘空间（推荐 100MB+）
