# 中国象棋 AI 对战

## 介绍

这是一个完整的中国象棋人机对战游戏，包含：
- 优美的 Pygame 界面
- **专业级 AI 算法**（参考象棋巫师）
- **完整的高级搜索算法**
- AI 思考过程展示

## 运行方法

```bash
python Chinese_chess.py
```

## AI 算法（基于 auto_chess2.py）

### 核心搜索算法
- **PVS（主要变例搜索）** - 高效的搜索算法
- **Alpha-Beta 剪枝** - 经典的高效搜索算法
- **迭代加深搜索** - 逐步加深搜索深度

### 高级优化技术
- **置换表** - 记忆化搜索，避免重复计算
- **杀手走法** - 保留每层的最佳走法
- **历史表** - 记录好的走法，分数为深度的平方
- **MVV/LVA 排序** - 吃子着法优先排序

### 高级功能
- **静态搜索（Quiescence Search）** - 避免水平线效应
- **将军延伸** - 被将军时自动加深搜索
- **重复局面判断** - 检测长将和循环
- **开局表** - 标准开局走法

### 评估函数
- **子力价值**（参考象棋巫师）：
  - 将/帅：10000
  - 车：1000
  - 马/炮：450
  - 相/象/士/仕：200
  - 兵/卒：100
- **位置价值矩阵** - 每个位置有不同的加成
- **窝心马惩罚** - 检测并惩罚窝心马

## 算法原理（参考 README）

### PVS 搜索
主要变例搜索（Principal Variation Search）是一种高效的 Alpha-Beta 变种，通过零窗口搜索来快速找到最佳走法。

### 置换表
- 键：棋盘局面 + 下棋方
- 值：最佳走法、分数、深度、节点类型（alpha/beta/pv）
- 分数修正：胜利分数需要根据深度调整

### 静态搜索
- 搜索深度为0时进入
- 只评估吃子走法
- 被将军时搜索所有走法
- 防止水平线效应

### 节点类型
- **Alpha 节点**：分数不超过下界
- **Beta 节点**：分数不低于上界
- **PV 节点**：分数在范围内
