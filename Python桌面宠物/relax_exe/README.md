# 定时休息

一个帮助你定时休息、保护眼睛健康的Windows桌面应用。

## 功能特性

- ⏰ **自定义时间**：可自由设置工作时长和休息时长
- 🔒 **强制锁屏休息**：工作时间结束后自动锁定计算机
- 🔴 **最后1分钟提醒**：倒计时最后1分钟字体变红提醒
- 🚫 **休息期间锁定**：休息期间强制锁定，禁止提前解锁
- 🖥️ **桌面右下角显示**：倒计时窗口显示在屏幕右下角，可拖动可调整大小
- 📜 **用户协议确认**：每次启动需确认用户协议

## 安装依赖

```bash
pip install -r requirements.txt
```

## 快速开始

```bash
python main.py
```

## 使用说明

### 1. 用户协议
每次启动应用时，会先显示用户协议对话框：
- 点击 **同意** → 继续使用应用
- 点击 **拒绝** 或关闭窗口 → 退出应用

### 2. 主界面
- 倒计时窗口会显示在屏幕右下角
- 可拖动窗口到任意位置
- 可拖拽边缘调整窗口大小

### 3. 控制按钮
- **开始** → 开始工作倒计时
- **暂停** → 暂停倒计时
- **停止** → 停止倒计时并重置
- ⚙️ **设置** → 调整工作/休息时长

### 4. 工作流程
```
开始工作 → 倒计时 → 时间到 → 自动锁屏 → 休息倒计时 → 休息结束 → 继续工作
```

### 5. 强制锁定说明
**重要**：在休息期间，如果您尝试解锁计算机，软件会立即重新锁定屏幕，直到休息时间结束。

如遇特殊情况无法解锁，请：
- 方法一：等待休息时间结束
- 方法二：重启计算机（强制关机重启）

## 默认设置

- 工作时长：20 分钟
- 休息时长：5 分钟

可通过 ⚙️ 设置按钮随时修改。

## 技术栈

- Python 3.8+
- PySide6（Qt 6 GUI框架）
- pywin32（Windows API调用）

## 注意事项

1. **仅支持Windows系统**：本应用使用Windows特定API进行锁屏和会话检测
2. **休息期间请勿强行关机**：除非必要，否则请等待休息结束
3. **日志文件**：应用运行日志保存在 `logs/` 目录下，方便排查问题
