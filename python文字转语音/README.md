# 文字转语音工具

基于Tkinter的文字转语音工具，稳定可靠！

## 🔥 核心改进

### v2.1.0 稳定版
- ✅ **pyttsx3直接播放** - 优先使用最稳定的方式
- ✅ **双重保障** - 先直接播放，后台再生成文件
- ✅ **极简依赖** - 只需pyttsx3和gTTS两个库
- ✅ **自动检测安装** - 启动器自动检查环境

## 🚀 快速开始

### 方式一：使用启动器（推荐）

```bash
python 启动工具.py
```

启动器会自动检查依赖，需要的话会自动安装！

### 方式二：手动运行

1. **安装依赖**
```bash
pip install -r requirements.txt
```

2. **运行程序**
```bash
python main.py
```

## ✨ 功能特点

### 基础功能
- 📝 多行文本输入
- ▶️ 一键转换播放
- ⏹️ 停止播放控制
- 💾 保存音频文件

### 音频设置
- 🎚️ 三种音质选择
- ⏩ 0.5x-2.0x 语速调节
- 👤 男声/女声/中性声

### 智能文本优化（本地）
- 🔢 阿拉伯数字 → 中文数字
  - 123 → 一百二十三
  - 2024年 → 二零二四年
  - 3.14 → 三点一四
- 📝 标点符号统一和优化
- 📄 常见缩写自动展开
- 🧹 清理特殊字符

## 📖 使用说明

### 1. 文本转换
1. 在左侧输入框输入要转换的文字
2. （可选）调整音频设置
3. 点击「转换并播放」
4. （可选）点击「保存音频」保存文件

**⭐ 最佳实践：** 使用pyttsx3引擎，无需网络，最稳定！

### 2. 文本优化
1. 输入文本后，在右侧面板
2. 点击「🔄 优化文本」
3. 文本会自动优化得更适合朗读

## 🔧 引擎说明

### pyttsx3（强烈推荐）
- ✅ 离线使用，无需网络
- ✅ 直接播放，最稳定
- ✅ 响应速度快
- ✅ 中文支持良好
- ❌ 音质相对一般

### gTTS
- ✅ 音质更好（Google语音）
- ❌ 需要网络连接
- ❌ 依赖Google服务可用性

## ⚠️ 常见问题

### Q: 播放没有声音？
A: 请确保使用pyttsx3引擎（推荐），并检查系统音量。

### Q: 程序卡住？
A: 新版本已彻底解决！所有处理都在独立线程，不会卡住界面。

### Q: 如何保存音频？
A: 先用pyttsx3播放（会自动生成文件），然后点击「保存音频」。

### Q: gTTS无法使用？
A: gTTS需要网络，且依赖Google服务。推荐直接用pyttsx3。
